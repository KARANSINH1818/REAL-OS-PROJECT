File Operations

Explanation of various file operations in an Operating System:

1. Create

Creates a new file in the file system.

2. Delete

Removes a file from the file system.

3. Open

Opens an existing file for reading or writing.

4. Close

Closes an open file.

5. Read

Reads data from a file.

6. Write

Writes data to a file.

7. Append

Adds data to the end of a file.

8. Seek

Moves file pointer to a specific position.

9. Get & Set Attributes

Retrieves or modifies file properties.

10. Rename

Changes file name.
