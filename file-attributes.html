File Attributes

1. Name:

- Only information kept in human-readable form. A string of alphanumeric characters and symbols like (_) used by the user.

2. Identifier - Unique Tag:

- Uniquely identifies the file within the file system. Used by OS to refer to the file.

3. Type:

- Expressed as a file extension, e.g., prog.cpp indicates a C++ file.

4. Location:

- Pointer to the device and location on the device of the file.

5. Size:

- Specifies the size of the file in bytes, words, or blocks.

6. Protection:

- Determines who can read, write, or execute the file.

7. Usage Count:

- Indicates the number of processes currently using this file.
